# mysql数据库和SQL

* mysql
  * 关系型数据库：以关系的形式来组织数据。
  * 三大范式：
    * 第一原则性，列的属性不可分割
    * 消除部分函数依赖
    * 消除传递依赖
  * 数据分割成表，表中一个个元组映射实体

* 数据库表的设计
  * 数据类型的设计
    * 如何利用好内存空间
    * 如何设计表之间的关系
    * 如何设计主键

*  SQl ：
  * DML：增删改查
  * DDL：改变表的结构，数据类型，表之间的链接和约束等初始化工作
  * DCL： 权限

* 经验
  * 项目开始的时候设计好表的结构，类型
  * 后期改数据库的代价更大
